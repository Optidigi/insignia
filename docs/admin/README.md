# Dashboard (spec)

This folder contains the functional specification for the Shopify embedded admin dashboard.

## Canonical references

- Data schemas: [`../core/data-schemas.md`](../core/data-schemas.md)
- Admin API contract: [`../core/api-contracts/admin.md`](../core/api-contracts/admin.md)
